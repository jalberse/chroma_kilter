Chroma Kilter Build Instructions
================================

The program can be built and run using the included Gradle scripts, which are 
modifed versions of scripts written by Chris Weaver.

Steps:

1. Navigate to the Build directory, which contains the script gradlew. Run 
"gradlew installDist" at the command line to compile the program.

2. Navigate to the subdirectory build/install/base/bin. Run the script 
chroma_kilter at the command line to start the program.
