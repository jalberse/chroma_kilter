Chroma Kilter User Instructions
===============================

After compiling and running the program (see readme-build.txt), the program 
can be controlled with the following keys:

1. 'p'             Pause rotation

2. 'i', 'o'        Change rotation speed

3. 'k', 'l'        Change distance to object

4. 0-7	           Change scene

5. 'z', 'x', 'c'   Toggle render modes (nondestructive, basic, nodepth)

6. 'w'             Toggle wireframe

7. up, down        Increase/Decrease the displacement of the chromatic aberration

8. shift + up,     Increase/Decrease the transparency of the aberrations
   shift + down

9. left, right     Rotate aberration axis (counterclockwise, clockwise)

10. minus, equals  Toggle background color

11. shift + minus, Change object color
    shift + equals
